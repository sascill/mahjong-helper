# 패 읽는 법 기능 스펙

## 목적

초보자가 만수·통수·삭수·자패의 모양과 이름을 빠르게 대응할 수 있게 한다.

역 예시나 일반 패 표시는 실제 패 이미지를 그대로 사용하고, 읽는 이름은 이 기능에서만 캡션으로 제공한다.

## 공개 인터페이스

```ts
function TileReference(): JSX.Element;
```

외부 레이어는 기능 루트의 `index.ts`를 통해서만 컴포넌트를 가져온다.

## 화면 구성

다음 순서로 패 그룹을 표시한다.

1. 만수
2. 통수
3. 삭수
4. 자패

각 그룹은 패 이미지를 격자로 표시하고, 각 패 아래에 읽는 이름을 캡션으로 제공한다.

그룹 설명문은 노출하지 않고, 그룹 제목과 패 캡션만 표시한다.

캡션은 다음 형식을 사용한다.

- 만수: `1만`부터 `9만`
- 통수: `1통`부터 `9통`
- 삭수: `1삭`부터 `9삭`
- 자패: `동`, `남`, `서`, `북`, `백`, `발`, `중`

## 동작 규칙

- 모든 기본 패 34종을 표시한다.
- 패 이미지는 공용 `MahjongTile`을 사용한다.
- `MahjongTile`의 접근성 이름과 화면 캡션은 같은 읽는 이름을 사용한다.
- 패 읽는 법 화면은 손패 분석 상태와 독립적으로 직접 접근할 수 있다.

## 제외 범위

- 패 선택
- 손패 분석
- 조패 추천
- 적도라 표시

## 테스트 기준

- `/tiles` 경로에서 패 읽는 법 화면을 표시한다.
- 만수·통수·삭수·자패 그룹을 표시한다.
- 34종 패 이미지를 모두 표시한다.
- 각 패의 읽는 이름을 캡션으로 표시한다.
