# 용어 기능 스펙

## 목적

초보자가 마작 설명에서 자주 만나는 용어를 짧게 확인할 수 있게 한다.

이 기능은 룰 설명이나 손패 분석을 제공하지 않고, 용어 인덱스 역할만 한다.
용어 데이터는 도메인 `GLOSSARY_TERMS`와 조회 함수를 사용한다.

## 공개 인터페이스

```ts
function TermReference(): JSX.Element;
```

- 외부 레이어는 기능 루트의 `index.ts`를 통해서만 컴포넌트를 가져온다.

## 용어 목록

용어 화면의 제목은 `용어`로 표시한다.

용어는 초보자가 훑기 쉬운 그룹으로 나누어 표시한다.

- 기본 진행
- 패와 형태
- 역과 점수

각 용어에는 다음 정보를 표시한다.

- 용어 이름
- 한 줄 설명

용어 설명은 도메인 용어 데이터의 초보자용 설명을 표시하고, 자세한 룰 해설은 다루지 않는다.
기존 룰 정보 화면의 제목이나 룰 카드 목록은 노출하지 않는다.

## 화면 경로

- `/terms` 경로에서 용어 화면을 표시한다.
- `/rules` 경로는 용어 화면으로 이동한다.
- 하단 주요 메뉴에는 용어 경로를 표시한다.

## 제외 범위

- 상세 용어 페이지
- 검색과 필터
- 점수 계산
- 룰별 예외 설명
