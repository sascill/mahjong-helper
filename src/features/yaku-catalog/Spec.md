# 역 정보 기능 스펙

## 목적

현재 손패와 관계없이 지원하는 역의 목록을 탐색하고, 각 역의 판수·조건·완성 예시를 확인할 수 있게 한다.

역의 정보는 역 도메인의 `YAKUS`와 `Yaku` 데이터를 그대로 사용한다. 이 기능은 역의 성립 여부나 추천 거리를 계산하지 않는다.

## 공개 인터페이스

```ts
type YakuCatalogProps = {
  selectedYaku?: Yaku;
};

function YakuCatalog(props: YakuCatalogProps): JSX.Element;
```

- `selectedYaku`가 없으면 지원하는 역 목록을 표시한다.
- `selectedYaku`가 있으면 해당 역의 상세 정보를 표시한다.
- 외부 레이어는 기능 루트의 `index.ts`를 통해서만 컴포넌트를 가져온다.

## 역 목록

역 목록 화면의 제목은 `역 정보`로 표시한다.

역 목록은 `YAKUS`에 정의된 순서를 유지하면서 다음 구분으로 표시한다.

화면 상단은 제목만 표시하고, 별도의 보조 설명문은 노출하지 않는다.

제목 아래에는 1판부터 역만까지 이동하는 sticky 앵커 네비게이션을 제공한다. 각 앵커는 같은 화면 안의 판수별 섹션으로 스크롤한다.

- 1판
- 2판
- 3판
- 6판
- 역만

각 역에는 다음 정보를 표시한다.

- 역 이름
- 역 이름 줄 오른쪽의 멘젠 여부 또는 울었을 때의 판수
- 한 줄로 읽기 쉬운 축약 설명

멘젠 판수 또는 역만 여부는 판수별 섹션 제목으로 확인하게 하고, 목록 카드 안에는 반복 표시하지 않는다.
목록 카드의 하단에는 별도의 조건 영역이나 자세히 보기 문구를 두지 않는다.
멘젠 전용 역은 제목 줄 오른쪽에 `멘젠`, 울 수 있는 역은 `울면 2판`처럼 표시한다.

판수별 섹션 제목은 sticky로 표시한다. 현재 섹션 안에서는 제목이 상단에 붙고, 다음 판수 섹션을 만나면 다음 제목이 이어서 붙는다.

목록 카드의 축약 설명은 상세 설명과 별도로 관리한다. 축약 설명은 모바일에서 빠르게 훑을 수 있도록 짧은 명사형 또는 동작형 문구를 사용하고, `입니다` 같은 문장형 종결 표현은 사용하지 않는다.

각 역은 `/yaku/:yakuId` 상세 경로로 이동할 수 있어야 한다.

## 역 상세

상세 화면에는 다음 정보를 표시한다.

- 역 이름
- 간단한 설명
- 멘젠 판수 또는 역만
- 울었을 때의 판수, 울기 가능 여부 또는 울기 불가
- 성립 조건
- 멘젠 필수 여부
- 치·퐁·깡 가능 여부
- 마작패 이미지로 구성한 완성 예시

멘젠과 울기 조건은 `Yaku.value`를 기준으로 표시한다.

```text
일반 역의 open이 null
→ 멘젠 필수
→ 치·퐁·깡 불가

일반 역의 open이 숫자
→ 멘젠 필수 아님
→ 치·퐁·깡 가능

역만의 open이 false
→ 역만
→ 멘젠 필수
→ 치·퐁·깡 불가

역만의 open이 true
→ 역만
→ 멘젠 필수 아님
→ 치·퐁·깡 가능
```

완성 예시는 역 데이터의 묶음 구분을 유지한다. 각 패는 이미지와 읽을 수 있는 접근성 이름을 함께 제공한다.

## 화면 경로

```text
/yaku
→ 역 목록

/yaku/:yakuId
→ 역 상세
```

- 존재하는 `yakuId`는 해당 역 상세를 표시한다.
- 존재하지 않는 `yakuId`는 `/yaku`로 이동한다.
- 목록과 상세 화면은 분석 결과 없이 직접 접근할 수 있다.

## 애플리케이션 연결

- 홈 화면에서 역 목록으로 이동할 수 있어야 한다.
- 하단 주요 메뉴에 역 정보 경로를 표시한다.
- 하단 주요 메뉴에는 패 읽는 법과 룰 정보 경로도 함께 표시한다.
- 추천 결과 카드를 누르면 해당 역의 상세 경로로 이동한다.
- 추천 결과 화면의 분석 상태는 상세 화면으로 이동해도 유지한다.
- 추천 결과에서 상세 화면으로 이동할 때는 추천 결과에서 왔다는 탐색 상태를 함께 전달한다.
- 추천 결과에서 들어온 역 상세 화면은 고정된 공통 헤더에 `/recommendations`로 이동하는 뒤로가기 버튼을 표시한다.
- 역 목록이나 주소로 직접 들어온 역 상세 화면은 고정된 공통 헤더에 `/yaku`로 이동하는 뒤로가기 버튼을 표시한다.
- 역 상세 콘텐츠 내부에는 별도의 뒤로가기 버튼을 표시하지 않는다.

## 제외 범위

- 역 검색과 필터
- 세부 룰 설정
- 실제 점수 계산
- 현재 손패에서 역 성립 여부 판정

## 테스트 기준

- 목록에 지원하는 38개 역을 판수와 역만으로 구분해 표시한다.
- 목록에서 각 역의 상세 경로로 이동할 수 있다.
- 상세 화면에 이름, 설명, 판수와 성립 조건을 표시한다.
- 멘젠 전용 역은 멘젠 필수와 치·퐁·깡 불가로 표시한다.
- 울 수 있는 역은 멘젠과 울기 판수를 모두 표시한다.
- 역만은 판수 대신 역만으로 표시한다.
- 완성 예시를 데이터의 묶음 구분대로 표시한다.
- 추천 결과 카드에서 해당 역 상세로 이동한다.
- 존재하지 않는 역 ID는 역 목록으로 이동한다.
- 홈과 하단 주요 메뉴에서 역 목록으로 이동할 수 있다.
- 추천 결과에서 들어온 역 상세 화면의 공통 헤더에서 추천 결과로 이동할 수 있다.
- 그 외 역 상세 화면의 공통 헤더에서 역 목록으로 이동할 수 있다.
