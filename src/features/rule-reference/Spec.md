# 룰 정보 기능 스펙

## 목적

초보자가 게임 중 자주 헷갈리는 기본 룰을 짧게 확인할 수 있게 한다.

## 공개 인터페이스

```ts
function RuleReference(): JSX.Element;
```

외부 레이어는 기능 루트의 `index.ts`를 통해서만 컴포넌트를 가져온다.

## 화면 구성

기본 룰은 카드 목록으로 표시한다.

화면 상단은 제목만 표시하고, 별도의 보조 설명문은 노출하지 않는다.

각 카드에는 다음 정보를 제공한다.

- 룰 제목
- 한두 문장의 짧은 설명

## 초기 룰 항목

- 역이 있어야 화료할 수 있다.
- 도라는 역이 아니다.
- 리치는 멘젠 상태에서만 가능하다.
- 울면 일부 역은 성립하지 않거나 판수가 줄어든다.

## 테스트 기준

- `/rules` 경로에서 룰 정보 화면을 표시한다.
- 초기 룰 항목을 모두 표시한다.
