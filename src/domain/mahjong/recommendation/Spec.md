# 추천 도메인 스펙

## 목적

유효한 첫 손패 13장을 지원하는 역별로 평가하고, 화료형까지 필요한 패가 적은 역을 최대 3개 추천한다.

추천 결과는 사용자가 이번 판의 목표를 선택하기 위한 정보만 제공한다. 버릴 패, 실제 화료 확률과 이후 행동은 계산하지 않는다.

이 모듈은 React와 브라우저 API에 의존하지 않는 순수 TypeScript 코드로 작성한다.

## 입력

```ts
type HandInput = {
  tiles: TileId[];
  roundWind: Wind;
  seatWind: Wind;
};
```

- 손패 도메인의 `validateHandInput`을 통과한 값만 평가한다.
- 유효하지 않은 입력이 전달되면 오류를 발생시킨다.
- 장풍과 자풍은 이후 역패 평가에 사용한다. 현재 지원하는 세 역의 거리에는 영향을 주지 않는다.

## 추천 거리

추천 거리는 현재 손패에서 목표 역의 유효한 14장 화료형까지 새로 필요한 최소 패 수다.

현재 손패와 목표 화료형에서 같은 패를 가능한 많이 유지한다고 할 때 다음과 같이 계산한다.

```text
추천 거리
= 14
- 현재 손패와 목표 화료형에 공통으로 포함된 패의 최대 장수
```

같은 패는 실제 장수만큼만 공통 패로 계산한다.

예를 들어 현재 손패 13장 전체가 특정 화료형에 포함된다면 필요한 패는 1장이고 추천 거리는 1이다.

추천 거리는 다음 요소를 의미하지 않는다.

- 화료 확률
- 남은 순서 수
- 필요한 패의 산에 남은 장수
- 상대방의 손패와 버림패
- 구체적으로 버려야 할 패

## 화료형

역별 거리 계산에서는 해당 역이 성립하는 다음 화료형을 고려한다.

### 일반형

```text
몸통 4개 + 머리 1개
```

- 몸통은 슌쯔 또는 커쯔다.
- 같은 패는 최대 4장까지만 사용한다.

### 치또이츠형

```text
서로 다른 또이쯔 7개
```

- 같은 패 네 장을 두 개의 또이쯔로 사용하지 않는다.

탕야오와 혼일색은 일반형과 치또이츠형 중 더 가까운 거리를 사용한다.

치또이츠는 치또이츠형만 평가한다.

## 추천 결과

```ts
type RecommendationDistance = 1 | 2 | 3 | 4 | 5;

type Recommendation = {
  yakuId: YakuId;
  requiredTileCount: RecommendationDistance;
  reason: string;
};
```

- `yakuId`: 추천한 역의 식별자
- `requiredTileCount`: 해당 역의 화료형까지 필요한 최소 패 수
- `reason`: 현재 손패에서 확인한 특징을 설명하는 한 문장

내부 계산 점수, 필요한 패 목록과 버릴 패는 결과에 포함하지 않는다.

## 공개 인터페이스

```ts
function recommendYaku(input: HandInput): Recommendation[];
```

처리 순서는 다음과 같다.

1. 손패 입력을 검증한다.
2. 지원하는 모든 역을 개별 평가한다.
3. 추천 거리가 1~5인 결과만 남긴다.
4. 추천 거리가 가까운 순서로 정렬한다.
5. 거리가 같으면 고정 우선순위로 정렬한다.
6. 앞에서부터 최대 3개를 반환한다.

## 지원 역과 우선순위

첫 구현의 고정 우선순위는 다음과 같다.

```text
탕야오
→ 치또이츠
→ 혼일색
```

우선순위는 추천 거리가 같을 때만 사용한다.

### 탕야오

- 숫자패 2~8만 사용한 일반형과 치또이츠형을 평가한다.
- 이유에는 현재 손패의 1·9·자패 장수 또는 숫자패 2~8 중심이라는 특징을 사용한다.

### 치또이츠

- 전체 34종 패를 대상으로 치또이츠형을 평가한다.
- 이유에는 현재 손패의 또이쯔 개수를 사용한다.

### 혼일색

- 만수·통수·삭수마다 해당 숫자패와 자패만 사용한 화료형을 평가한다.
- 세 종류 중 추천 거리가 가장 가까운 숫자패를 선택한다.
- 거리가 같으면 만수·통수·삭수 순서로 선택한다.
- 이유에는 선택한 숫자패와 자패의 장수를 사용한다.

## 추천 제외

다음 결과는 추천하지 않는다.

- 추천 거리가 6 이상인 역
- 현재 지원하지 않는 역
- 유효하지 않은 손패에서 계산된 결과

추천 가능한 역이 3개보다 적으면 결과를 억지로 채우지 않는다.

## 테스트 기준

- 유효하지 않은 손패를 거부한다.
- 화료형의 일부인 13장 손패를 추천 거리 1로 계산한다.
- 탕야오는 일반형과 치또이츠형 중 가까운 거리를 사용한다.
- 치또이츠는 서로 다른 또이쯔 개수와 고유 패 종류를 반영한다.
- 혼일색은 가장 가까운 숫자패 종류를 선택한다.
- 추천 거리가 6 이상인 역을 제외한다.
- 결과를 추천 거리 오름차순으로 정렬한다.
- 거리가 같으면 고정 우선순위를 사용한다.
- 결과는 최대 3개다.
- 추천 이유에는 현재 손패의 실제 특징이 반영된다.
- 입력 배열을 변경하지 않는다.
