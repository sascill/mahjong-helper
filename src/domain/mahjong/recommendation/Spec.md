# 추천 도메인 스펙

## 목적

유효한 첫 손패 13장에서 역별 특징이 충분히 드러난 목표 역만 평가하고, 화료형까지 필요한 패가 적은 역을 최대 3개 추천한다.

추천 결과는 사용자가 이번 판의 목표를 선택하기 위한 정보만 제공한다. 버릴 패, 실제 화료 확률과 이후 행동은 계산하지 않는다.

이 모듈은 React와 브라우저 API에 의존하지 않는 순수 TypeScript 코드로 작성한다.

## 입력

```ts
type HandInput = {
  tiles: TileId[];
  roundWind: Wind;
  seatWind: Wind;
};
```

- 손패 도메인의 `validateHandInput`을 통과한 값만 평가한다.
- 유효하지 않은 입력이 전달되면 오류를 발생시킨다.
- 장풍과 자풍은 역패 평가에서 가치패를 결정하는 데 사용한다.

## 추천 거리

추천 거리는 현재 손패에서 목표 역의 유효한 14장 화료형까지 새로 필요한 최소 패 수다.

현재 손패와 목표 화료형에서 같은 패를 가능한 많이 유지한다고 할 때 다음과 같이 계산한다.

```text
추천 거리
= 14
- 현재 손패와 목표 화료형에 공통으로 포함된 패의 최대 장수
```

같은 패는 실제 장수만큼만 공통 패로 계산한다.

예를 들어 현재 손패 13장 전체가 특정 화료형에 포함된다면 필요한 패는 1장이고 추천 거리는 1이다.

추천 범위를 벗어난 4장 이상의 거리는 내부에서 `4`로 정규화한다. 이 값은 정확한 장수를 사용자에게 보여주기 위한 값이 아니라 추천 제외를 위한 경곗값이다.

추천 거리는 다음 요소를 의미하지 않는다.

- 화료 확률
- 남은 순서 수
- 필요한 패의 산에 남은 장수
- 상대방의 손패와 버림패
- 구체적으로 버려야 할 패

## 화료형

역별 거리 계산에서는 해당 역이 성립하는 다음 화료형을 고려한다.

### 일반형

```text
몸통 4개 + 머리 1개
```

- 몸통은 슌쯔 또는 커쯔다.
- 같은 패는 최대 4장까지만 사용한다.

### 치또이츠형

```text
서로 다른 또이쯔 7개
```

- 같은 패 네 장을 두 개의 또이쯔로 사용하지 않는다.

탕야오와 혼일색은 일반형과 치또이츠형 중 더 가까운 거리를 사용한다.

치또이츠는 치또이츠형만 평가한다.

## 추천 결과

```ts
type RecommendationDistance = 1 | 2 | 3;

type Recommendation = {
  yakuId: YakuId;
  requiredTileCount: RecommendationDistance;
  reason: string;
};

type YakuEvaluation = {
  yakuId: YakuId;
  requiredTileCount: number;
  reason: string;
};
```

- `yakuId`: 추천한 역의 식별자
- `requiredTileCount`: 해당 역의 화료형까지 필요한 최소 패 수
- `reason`: 현재 손패에서 확인한 특징을 설명하는 한 문장
- `YakuEvaluation.requiredTileCount`의 `4`는 추천 범위를 벗어난 4장 이상을 뜻한다.

내부 계산 점수, 필요한 패 목록과 버릴 패는 결과에 포함하지 않는다.

## 공개 인터페이스

```ts
function evaluateYaku(
  input: HandInput,
  yakuId: YakuId,
): YakuEvaluation | null;

function recommendYaku(input: HandInput): Recommendation[];
```

`evaluateYaku`는 하나의 역을 평가한다.

- 첫 손패의 패 구성으로 평가할 수 있는 역은 거리와 이유를 반환한다.
- 현재 입력만으로 평가할 수 없는 역은 `null`을 반환한다.
- 유효하지 않은 손패는 오류를 발생시킨다.

`recommendYaku`의 처리 순서는 다음과 같다.

1. 손패 입력을 검증한다.
2. 손패에서 추천에 필요한 공통 특징을 한 번 계산한다.
3. 목표 역별 최소 특징을 만족하는 역만 남긴다.
4. 남은 목표 역의 추천 거리를 계산한다.
5. 추천 거리가 1~3인 결과만 남긴다.
6. 추천 거리가 가까운 순서로 정렬한다.
7. 거리가 같으면 고정 우선순위로 정렬한다.
8. 앞에서부터 최대 3개를 반환한다.

## 추천 역할

전체 역은 추천에서 다음 역할 중 하나를 가진다.

- `primary`: 첫 손패에서 직접 추천할 수 있는 목표 역
- `upgrade`: 독립 추천하지 않고 이후 발전 가능한 역
- `catalog-only`: 첫 손패 추천에 사용하지 않는 역

### 목표 역

```text
탕야오 → 핑후 → 역패 → 치또이츠 → 또이또이
→ 혼일색 → 일기통관 → 삼색동순 → 찬타 → 국사무쌍
```

위 순서는 추천 거리가 같을 때만 사용하는 고정 우선순위다.

### 발전 역

이페코, 삼색동각, 산안커, 혼노두, 소삼원, 량페코, 준찬타, 청일색, 구련보등, 녹일색, 스앙커, 청노두, 자일색, 대삼원, 소사희와 대사희는 거리를 계산할 수 있지만 독립된 첫 손패 목표로 추천하지 않는다.

### 도감 전용 역

리치, 일발, 멘젠쯔모, 창깡, 영상개화, 해저로월, 하저로어, 더블리치, 산깡쯔, 천화, 지화와 스깡쯔는 선언·화료 과정·깡 정보가 필요해 거리 평가와 추천을 하지 않는다.

## 손패 특징

추천 정책은 손패를 반복해서 분석하지 않고 다음 공통 특징을 한 번 계산해 사용한다.

- 정확히 두 장인 또이쯔 개수
- 두 장 이상인 또이쯔·커쯔 후보 개수
- 세 장 이상인 커쯔 개수
- 숫자패 2~8과 1·9·자패의 장수
- 만수·통수·삭수와 자패의 장수
- 두 장 이상 모인 가치패 종류 수
- 두 장 이상 이어진 슌쯔 후보 수
- 한 무늬의 123·456·789 구간 후보 수
- 같은 숫자 구간이 있는 무늬 수
- 123·789 슌쯔 후보 수
- 서로 다른 1·9·자패 종류 수

## 목표 역별 최소 특징

최소 특징을 만족하지 않는 역은 거리가 3 이하더라도 추천하지 않는다.

| 목표 역 | 추천에 필요한 최소 특징 |
| --- | --- |
| 탕야오 | 1·9·자패가 3장 이하 |
| 핑후 | 슌쯔 후보가 3개 이상이고 또이쯔·커쯔 후보가 1개 이상 |
| 역패 | 같은 가치패가 2장 이상 |
| 치또이츠 | 정확히 두 장인 또이쯔가 4개 이상 |
| 또이또이 | 또이쯔·커쯔 후보가 3개 이상 |
| 혼일색 | 한 무늬가 5장 이상이고 해당 무늬와 자패가 합계 9장 이상이며 자패가 1장 이상 |
| 일기통관 | 한 무늬에서 123·456·789 구간 중 2개 이상이 슌쯔 후보 |
| 삼색동순 | 같은 숫자 구간이 2가지 이상의 무늬에서 슌쯔 후보 |
| 찬타 | 1·9·자패가 5장 이상이고 123·789 슌쯔 후보가 1개 이상 |
| 국사무쌍 | 서로 다른 1·9·자패가 9종 이상 |

## 역별 거리 규칙

- 일반형 역은 몸통과 머리 조건을 만족하는 모든 14장 화료형 중 최소 거리를 사용한다.
- 치또이츠가 성립할 수 있는 탕야오·혼일색·청일색·혼노두·자일색은 일반형과 치또이츠형 중 최소 거리를 사용한다.
- 핑후는 슌쯔 네 개와 가치패가 아닌 머리를 목표 형태로 사용한다. 양면대기 여부는 실제 화료 단계의 추가 조건으로 남는다.
- 역패는 백·발·중, 장풍 또는 자풍의 커쯔가 포함된 일반형을 평가한다.
- 산안커와 스앙커는 첫 손패에서 멘젠을 유지한다는 전제로 각각 커쯔 세 개와 네 개가 포함된 형태를 평가한다.
- 국사무쌍과 구련보등은 각 특수 화료형을 직접 평가한다.
- 한 손패가 여러 역의 목표 형태에 동시에 가까우면 각 역을 독립적으로 평가한다.

## 추천 제외

다음 결과는 추천하지 않는다.

- 목표 역별 최소 특징을 만족하지 않는 역
- 추천 거리가 4 이상인 역
- 발전 역과 도감 전용 역
- 유효하지 않은 손패에서 계산된 결과

추천 가능한 역이 3개보다 적으면 결과를 억지로 채우지 않는다.

## 테스트 기준

- 유효하지 않은 손패를 거부한다.
- 38개 전체 역이 목표·발전·도감 전용 중 하나로 빠짐없이 구분된다.
- 거리 평가가 가능한 모든 역은 해당 역의 완성 예시에서 한 장을 뺀 손패를 거리 1로 계산한다.
- 상황·화료 과정·깡 선언 정보가 필요한 역은 `null`로 평가한다.
- 목표 역별 최소 특징을 만족하지 않으면 거리가 3 이하여도 추천하지 않는다.
- 발전 역은 거리가 1이어도 직접 추천하지 않는다.
- 화료형의 일부인 13장 손패를 추천 거리 1로 계산한다.
- 탕야오는 일반형과 치또이츠형 중 가까운 거리를 사용한다.
- 치또이츠는 서로 다른 또이쯔 개수와 고유 패 종류를 반영한다.
- 혼일색은 가장 가까운 숫자패 종류를 선택한다.
- 추천 거리가 4 이상인 역을 제외한다.
- 결과를 추천 거리 오름차순으로 정렬한다.
- 거리가 같으면 고정 우선순위를 사용한다.
- 결과는 최대 3개다.
- 추천 이유에는 현재 손패의 실제 특징이 반영된다.
- 입력 배열을 변경하지 않는다.
