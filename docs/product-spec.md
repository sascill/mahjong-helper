# 제품 스펙

## 목적

리치마작 초보자가 첫 손패를 받은 뒤 매번 역 목록을 찾아보지 않고, 이번 판에서 목표로 삼을 역을 빠르게 선택하도록 돕는다.

앱은 목표만 제시하며 구체적으로 어떤 패를 버릴지는 알려주지 않는다.

## 대상 사용자

- 리치마작의 기본 진행 방법은 알고 있는 초보자
- 역의 종류와 성립 조건이 아직 익숙하지 않은 플레이어
- 게임 흐름을 끊지 않고 첫 손패의 방향만 정하고 싶은 플레이어

## 핵심 사용 흐름

```text
첫 손패 13장 선택
→ 장풍과 자풍 선택
→ 가까운 역 최대 3개 확인
→ 목표로 삼을 역 선택
→ 역의 조건과 완성 예시 확인
→ 직접 플레이
```

## MVP 기능

### 손패 입력

- 만수·통수·삭수·자패 중 13장을 선택한다.
- 같은 패는 최대 4장까지 선택할 수 있다.
- 선택한 패를 취소하거나 전체 손패를 초기화할 수 있다.
- 선택한 손패는 정해진 순서로 자동 정렬한다.
- 장풍과 자풍을 선택한다.

### 역 추천

- 현재 손패에서 화료까지 필요한 패가 적은 역을 찾는다.
- 필요한 패가 1장부터 5장까지인 역만 추천한다.
- 가까운 순서대로 최대 3개를 보여준다.
- 추천할 역이 부족하면 3개를 억지로 채우지 않는다.
- 각 추천에는 역 이름, 판수, 필요한 최소 장수와 추천 이유를 표시한다.
- 버릴 패, 화료 확률과 내부 추천 점수는 표시하지 않는다.

### 역 도감

- 지원하는 역의 목록을 제공한다.
- 역 이름, 판수, 설명과 성립 조건을 제공한다.
- 멘젠 필수 여부와 치·퐁·깡 가능 여부를 표시한다.
- 유니코드 마작패로 완성 예시를 보여준다.

## 화면

1. 홈
2. 손패 선택
3. 추천 결과
4. 역 목록
5. 역 상세

추천 결과와 역 목록은 동일한 역 상세 화면을 사용한다.

## MVP 제외 범위

- 사진 및 카메라 인식
- 버릴 패 추천과 실시간 조패 안내
- 실제 화료 확률
- 후리텐 판정
- 도라, 부수와 정확한 점수 계산
- 점수 관리
- 게임 기록과 통계
- 세부 룰 설정
- 온라인 대전

## 완료 기준

- 사용자가 유효한 첫 손패 13장과 장풍·자풍을 입력할 수 있다.
- 입력한 손패를 기준으로 가까운 역을 최대 3개 확인할 수 있다.
- 추천된 역의 기본 조건과 완성 예시를 확인할 수 있다.
- 역 도감에서 지원하는 역을 탐색할 수 있다.
- 모바일 화면에서 모든 핵심 기능을 사용할 수 있다.
