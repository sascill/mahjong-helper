# 제품 스펙

## 문서 목적

Mahjong Helper MVP가 사용자에게 제공해야 하는 기능과 완료 조건을 정의한다.

구현 기술, 프로젝트 구조와 추천 계산 알고리즘은 이 문서에서 다루지 않는다.

## 제품 목적

리치마작 초보자가 첫 손패를 받은 뒤 역 목록을 반복해서 확인하지 않고, 이번 판에서 목표로 삼을 역을 빠르게 선택하도록 돕는다.

앱은 목표가 될 역만 제시한다. 구체적으로 어떤 패를 버릴지 알려주거나 플레이를 대신하지 않는다.

## 대상 사용자

- 리치마작의 기본 진행 방법을 알고 있는 초보자
- 역의 종류와 성립 조건이 익숙하지 않은 플레이어
- 첫 손패에서 어떤 역을 목표로 해야 할지 판단하기 어려운 플레이어

## 핵심 사용 흐름

```text
첫 손패 13장 선택
→ 장풍과 자풍 선택
→ 손패 분석
→ 가까운 역 최대 3개 확인
→ 목표로 삼을 역 선택
→ 역의 조건과 완성 예시 확인
→ 직접 플레이
```

## 기준 룰

MVP는 다음 고정 룰을 사용한다.

- 쿠이탕을 허용한다.
- 인화는 사용하지 않는다.
- 유국만관은 사용하지 않는다.
- 국사무쌍 13면 대기, 스앙커 단기, 순정구련보등과 대사희를 더블 역만으로 취급하지 않는다.
- 서로 다른 역만이 함께 성립하면 각각의 역만을 모두 인정한다.

룰 설정 화면과 사용자별 룰 변경은 MVP에서 제공하지 않는다.

## 기능 요구사항

### P-01 패 선택

사용자는 만수·통수·삭수·자패 중에서 첫 손패 13장을 선택할 수 있어야 한다.

- 같은 패는 최대 4장까지 선택할 수 있다.
- 선택한 패를 개별적으로 취소할 수 있다.
- 선택한 패를 전체 초기화할 수 있다.
- 선택한 손패는 정해진 패 순서로 자동 정렬한다.
- 현재 선택한 패의 수를 표시한다.

### P-02 장풍과 자풍

사용자는 손패 분석 전에 장풍과 자풍을 선택해야 한다.

- 장풍은 동·남·서·북 중에서 선택한다.
- 자풍은 동·남·서·북 중에서 선택한다.
- 장풍과 자풍은 역패와 관련된 판정에 사용한다.

### P-03 입력 검증

다음 조건을 모두 만족해야 손패를 분석할 수 있다.

- 손패가 정확히 13장이다.
- 같은 패가 4장을 초과하지 않는다.
- 장풍이 선택되어 있다.
- 자풍이 선택되어 있다.

조건을 만족하지 않으면 분석을 실행하지 않고 원인을 사용자에게 보여준다.

### P-04 역 추천

앱은 현재 손패에서 화료까지 필요한 패가 적은 역을 찾는다.

- 첫 손패의 패 구성만으로 목표를 평가할 수 있는 역만 추천한다.
- 상황과 화료 과정으로만 성립하는 역은 도감에 포함하되 추천하지 않는다.
- 필요한 패가 1장부터 5장까지인 역만 추천한다.
- 가까운 순서대로 최대 3개를 보여준다.
- 추천할 역이 부족하면 3개를 억지로 채우지 않는다.
- 추천 계산 규칙은 `recommendation-spec.md`를 기준으로 한다.

### P-05 추천 결과

각 추천 결과에는 다음 정보를 표시한다.

- 역 이름
- 판수
- 완성까지 필요한 최소 장수
- 추천 이유

다음 정보는 표시하지 않는다.

- 버릴 패
- 실제 화료 확률
- 내부 추천 점수
- 이후 순서별 행동 안내

### P-06 역 상세

사용자는 추천 결과에서 역을 선택해 상세 정보를 확인할 수 있어야 한다.

상세 화면에는 다음 정보를 표시한다.

- 역 이름
- 판수
- 간단한 설명
- 성립 조건
- 멘젠 필수 여부
- 치·퐁·깡 가능 여부
- 유니코드 마작패로 구성한 완성 예시

### P-07 역 목록

사용자는 현재 손패와 관계없이 기준 룰에서 사용하는 모든 일반 역과 역만을 확인할 수 있어야 한다.

역 목록에서 역을 선택하면 추천 결과와 동일한 역 상세 화면으로 이동한다.

- 판수별 일반 역과 역만을 구분한다.
- 첫 손패 추천 대상이 아닌 상황역도 목록과 상세에 포함한다.
- 인화와 유국만관은 기준 룰에서 사용하지 않으므로 목록에서 제외한다.

## 화면 구성

### 홈

- 첫 손패 분석 진입
- 역 목록 진입

### 손패 선택

- 손패 13장 선택
- 장풍 선택
- 자풍 선택
- 입력 초기화
- 분석 실행

### 추천 결과

- 입력한 손패 표시
- 추천 역 최대 3개 표시
- 역 상세 진입

### 역 목록

- 지원하는 역 표시
- 역 상세 진입

### 역 상세

- 역의 판수, 설명, 조건과 완성 예시 표시

## MVP 제외 범위

- 사진 및 카메라 인식
- AI 모델 학습
- 버릴 패 추천
- 실시간 조패 안내
- 실제 화료 확률
- 후리텐 판정
- 도라 계산
- 부수와 정확한 점수 계산
- 점수 관리
- 게임 기록과 통계
- 세부 룰 설정
- 온라인 대전

## 완료 조건

다음 조건을 모두 만족하면 제품 MVP가 완료된 것으로 판단한다.

- 사용자가 유효한 첫 손패 13장을 입력할 수 있다.
- 사용자가 장풍과 자풍을 입력할 수 있다.
- 유효하지 않은 입력으로 분석할 수 없다.
- 현재 손패에서 가까운 역을 최대 3개 확인할 수 있다.
- 추천된 역의 상세 조건과 완성 예시를 확인할 수 있다.
- 손패 분석 없이도 역 목록과 상세 정보를 확인할 수 있다.
- 기준 룰에서 사용하는 모든 일반 역과 역만을 도감에서 확인할 수 있다.
- 모바일 화면에서 모든 핵심 기능을 사용할 수 있다.
