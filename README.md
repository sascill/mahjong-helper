# Mahjong Helper

리치마작 초보자가 첫 손패를 받고 이번 판에 노려볼 만한 역을 빠르게 선택할 수 있도록 돕는 프로젝트입니다.

## 프로젝트 소개

마작 초보자는 첫 손패를 받은 뒤 어떤 역을 목표로 패를 만들어야 할지 판단하기 어렵습니다.

패를 받을 때마다 역 목록을 다시 확인하면 게임의 흐름이 끊기고, 게임을 즐기기보다 규칙을 공부하는 시간이 길어집니다.

Mahjong Helper는 첫 손패를 한 번 입력하면 현재 손패에서 가까운 역을 제시하여, 플레이어가 이번 판의 목표를 빠르게 정할 수 있도록 돕습니다.

> 앱은 목표를 제시하고, 실제 조패와 판단은 플레이어가 직접 합니다.

## 핵심 사용 흐름

```text
첫 손패 13장과 장풍·자풍 선택
→ 가까운 역 최대 3개 확인
→ 목표로 삼을 역 선택
→ 역의 조건과 완성 형태 확인
→ 직접 플레이
```

## 주요 기능

- 첫 손패 13장 선택
- 장풍·자풍 선택
- 현재 손패에서 가까운 역 추천
- 역의 조건과 완성 형태 확인
- 전체 역 목록 탐색
